Portegys Image Transform.

Transform an image for enhanced analysis.

The mean of the X coordinates of a selected set of image pixels is
used to create a linear vertical target line. Pixel rows are then rectified by
horizontally shifting their selected pixel to the mean.

Adapted from JavaPixelManipulation.java.

Reference:
Morphogenetic processes as data: Quantitative structure in the Drosophila eye imaginal disc
Bradly Alicea, Thomas E. Portegys, Diana Gordon, Richard Gordon
Biosystems. Volume 173, November 2018, Pages 256-265
doi: https://doi.org/10.1101/395640
